Overview

This repository contains the full security architecture documentation suite for a banking institution. The architecture implements defence-in-depth across 14 security zones, from the internet perimeter through to the data centre and disaster recovery site.

Architecture zones

Internet zone
External users, browsers, mobile clients
Perimeter security
ISP router, border router, DDoS, perimeter FW
DMZ network
WAF, reverse proxy, load balancer, DNS, email GW
DMZ web servers
Web server 172.16.10.20, API gateway .30
Internal firewall
172.16.21.10 — DMZ-to-app boundary
Application services
Banking app, mobile backend, IAM, MFA
Identity services
AD, DNS, RADIUS, DHCP — 172.16.60.0/24
Core network
Dual core routers + dual core switches
VLAN segmentation
HR / Finance / IT / SOC / Server Mgmt
ATM network
Isolated PCI-scoped ATM infrastructure
Payment processing
Payment GW, card processing, transactions
SOC
Wazuh SIEM, log collector, SOC dashboard
Data centre
Dual-firewall, core banking DB, storage, backup
Disaster recovery
DR site with replication from DC (RTO 4h / RPO 1h)
